# 配置项目

1.安装eslint-plugin-prettier

npm install eslint-plugin-prettier --save-dev

2.安装@vue/eslint-config-prettier

npm install @vue/eslint-config-prettier --save-dev

然后在package.json或.eslintrc.js或者其它设置eslint的地方，加上：@vue/prettier

"extends": [
  "plugin:vue/essential",
  "eslint:recommended",
  "@vue/prettier"
],

3.配置prettier
在项目的根目录下创建一个.prettierrc.js文件，然后在其中加入：

module.exports = {
  semi: false,
  singleQuote: true
}

4.npm run lint

# 配置VsCode

在VsCode中实时看到对错误的标注，实现Ctlr+S(autoFixOnSave)自动格式化保存(避免重复npm run lint)，配置如下。

1.安装插件eslint

2.在项目的根目录下创建一个.vscode文件夹，在.vscode文件夹新建settings.json。代码如下：

{
  "eslint.autoFixOnSave": true,
  "eslint.validate": [  
    "javascript",
    "javascriptreact",
    {
      "language": "vue",
      "autoFix": true
    }
  ],
}
